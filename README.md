PROMPROM
========

[![Build Status](https://travis-ci.org/davinche/promprom.svg?branch=master)](https://travis-ci.org/davinche/promprom)

A lightweight promise polyfill.


## Simple Usage ##

```js
import Promise from 'promprom';
```

## Polyfill ##

```js
import { polyfill } from 'promprom';
polyfill();
```
